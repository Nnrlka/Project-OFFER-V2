# Images Directory

Place your images here:

1. **Product images** - for product cards
2. **User avatars** - for profile pictures  
3. **Logos** - app logo and favicon
4. **Backgrounds** - background images for sections

Recommended sizes:
- Product images: 400x300px
- Avatars: 200x200px
- Logos: 512x512px (for favicon: 64x64px)

Formats: PNG, JPG, WebP
